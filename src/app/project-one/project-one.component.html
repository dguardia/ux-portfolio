Case: UX portfolio

Trying to apply for a new position is not always easy. As a UX designer, there needs to be a portfolio to showcase an ability to do the job. Transitioning from one position to another that has different qualifications requires proof that the UX designer has the tools and skills to do the new job. For this reason, I have created a solution that will help to display my skills by creating a case study of how does a professional UX designer put a portfolio together.

Problem: The UX designer professional needs to showcase their portfolio but they do not want to use a cookie cutter company website or design builder.

Solution: Build a custom application that will be easy to deploy and update portfolio items.
