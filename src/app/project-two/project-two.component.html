Case: UX Reputation System

Investment in education has become quite a preference. However, investment requires being in a right place. Also, while using an app, users feel comforts and discomforts about few features in it. And yes, this project is about taking in the comfortable features and refining the features that require attention and adding new features that are missing in the existing apps. This app, created by the best developers, is about the Reputation system that gives the user the better idea and user experience. The development was done by taking in the real-life personas as test subjects that will use the app and tailored to their interest. So, the app could fit in the requirements of the common users as well.

Problem: Students and other users of the app face hard time about picking the right professors or school due to wrong information and reviews. Also, some features of the apps are not user friendly and uneasy to use.

Solution: This app was developed with the intention to give better user design and authenticity by using the test subjects and their inputs as the key. This makes the reputation system more reliable and easy to use.

Full DisclousureThis UX project was done with a group of Graduate Stundents from NJIT from the class UX User Experience Design - IS667, year 2017. Also this UX process was authorized by the School and professor to be use as portfolio.

Problem Statement

Objective: To alleviate some of the following frustrations being faced by Persona I and II which represents parents and students in a broader sense:

Frustrations faced by Persona I

  • Ads that takes more screen space.
  • Too much of graphics that slows down web page.
  • More anonymous reviews that is misleading.
  • Does not have both Desktop and Mobile app version.
  • Too much of space to install on mobile as App.

Frustrations faced by Persona II

  • Parents are out of the U.S. so they are concerned about student's performance.
  • No easy UI on other apps that could give her a good navigation.
  • No demo videos of professor's lecture and campus.
  • Unable to access the updates about universities.
  • Access to the bio data of each professor.

This application was built to assist parents and students in alleviating the frustrations faced when looking for a good school and choosing a professor that will help to foster a studentâ€™s educational goals and career objectives.
